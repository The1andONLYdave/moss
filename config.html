This section lists various options you can make to your configuration file. Some of them can be overridden from the wallpaper settings activity.

Variable Description

color_add

 Added or override colors to color dictionary

background_color

 The background color for the wallpaper.

background_image

 An absolute path to a background image. /sdcard/moss/bg-fav.png

background_mod

 Like xsetroot's mod parameter, allows the user to create a plaid like pattern.

default_color

 Set the default color of the paint.

default_outline_color

 default outline color.

default_shade_color

 default shade color.

uppercase

 All text in uppercase

disable_auto_reload

 Enable to disable file observer (inotify) based auto config reload.

font

 Sets the default font for the display.

gap_x

 Additional horizonal spacing

gap_y

 Additional vertical spacing

update_interval

 Determines how frequently the data, and display are updated.
