--- layout: default title: Moss ---

Moss is a Conky like live wallpaper for android. It provides system statistics such as uptime, cpu usage, network usage, top processes, and battery level. It is highly configurable.

Moss ships with a few sample configurations. However you can write your own to customize your wallpaper. Configuration tries to closely follow Conky's syntax. Currently only a small subset of Conky's options are available.

For example configurations please see the samples.

Moss is free and open source. You can download the code and build it yourself from Github.
